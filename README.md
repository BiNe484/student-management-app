## Summary
**﻿Sản phẩm nhóm - Ứng dụng quản lý sinh viên và nhân sự** dành cho các vai trò **Admin**, **Manager**, và **Employee**, được xây dựng trên nền tảng Android sử dụng **Firebase** làm backend.

**Chức năng chính:**

- Authentication
- Quản lý người dùng
- Quản lý sinh viên
- Quản lý lịch sử đăng nhập
- Hồ sơ cá nhân (Profile)

**Công nghệ sử dụng**
- Java (Android)
- Firebase Authentication
- Firebase Realtime Database
- Firebase Storage
- Glide
- Material Design
